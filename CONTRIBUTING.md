# Contributing

Thanks for contributing to `sfb`.

## Development setup

1. Install dependencies:
   - `brew install fzf trash`
   - `brew install bats-core shellcheck` (for local checks)
2. Run checks:
   - `make lint`
   - `make test`

## Pull request expectations

- Keep PRs focused.
- Include tests for behavior changes.
- Update docs when behavior or flags change.
- Follow the PR template checklist.

## Security

Do not disclose vulnerabilities in public issues.
Use the Security tab / private advisory flow described in `.github/SECURITY.md`.
